# unix_terminal_commands    

# Unix Commands Cheatsheet    

## **ls**   
*Lists contents of a directory.*   
    
    Usage:
    Without arguments: Lists contents of current directory   
    With arguments: Lists contents of given directory   
    
    Example Usage:
    ls
    ls ~/
    ls -a    
    
    
 ## **cd**   
*Changes directory.*   

    Usage:   
    Without parameters: Changes to user's home directory.   
    With an argument: Changes to given directory.   
    
    Example Usage:   
    cd
    cd test  
    
    
 ## **pwd**   
*Path to Working Folder.*   

    Usage:   
    Always without arguments: Checks what is the current folder we are in.   
       
    Example Usage:   
    pwd
   


pwd is command that shows you the current directory
